# 🎮 Предсказание победы игрока (Нейросеть + GUI)

Этот проект представляет собой небольшое приложение на Python, которое использует **нейронную сеть (Keras/TensorFlow)** для предсказания вероятности победы игрока на основе его характеристик, а также предоставляет **графический интерфейс (Tkinter)** для ввода данных и получения результата.

## 🎯 Цель проекта
Разработать прототип системы, которая анализирует физические и навыковые параметры спортсмена и предсказывает, выиграет ли он матч, с последующей возможностью использования этой модели в реальном времени через удобное приложение.

## 🛠 Используемые технологии и библиотеки
- **Язык:** Python 3
- **Машинное обучение:**
  - `pandas` (обработка табличных данных)
  - `scikit-learn` (`train_test_split`, `StandardScaler` для нормализации)
  - `tensorflow.keras` (создание и обучение нейронной сети)
- **Интерфейс:** `tkinter` (стандартная библиотека для создания оконных приложений)

## 🧠 Архитектура модели
Модель представляет собой многослойный персептрон (MLP) для задачи **бинарной классификации** (победа / поражение).

- **Входной слой (Input):** 4 признака (Возраст, Опыт, Навык, Физическая форма).
- **Скрытые слои:**
  - Слой 1: 32 нейрона, функция активации `ReLU`.
  - Слой 2: 16 нейронов, функция активации `ReLU`.
  - Слой 3: 8 нейронов, функция активации `ReLU`.
- **Выходной слой (Output):** 1 нейрон, функция активации `Sigmoid` (выдает вероятность от 0 до 1).
- **Компиляция:** Оптимизатор `adam`, функция потерь `binary_crossentropy`.
- **Обучение:** 50 эпох, валидация на тестовой выборке (20% от всех данных).

## 📊 Данные
Модель обучается на синтетическом датасете, содержащем 7 записей со следующими полями:
- `age` (Возраст)
- `experience` (Опыт в годах)
- `skill` (Уровень навыка, 0-100)
- `fitness` (Физическая подготовка, 0-100)
- `win` (Целевая переменная: 1 — победа, 0 — поражение)

## 💻 Как использовать приложение

1. Запустите скрипт Python.
2. В открывшемся окне приложения заполните 4 поля:
   - **Age (Возраст)**
   - **Experience (Опыт)**
   - **Skill (Навык)**
   - **Fitness (Форма)**
3. Нажмите кнопку **"Predict Win"**.
4. Всплывающее окно покажет **вероятность победы в процентах** на основе обученной модели.
